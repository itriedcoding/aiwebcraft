## Develop

- Copy `.env.example` to `.env` and fill keys
- Start services: `docker compose up -d`
- Install deps: `pnpm install`
- Dev: `pnpm dev`

This monorepo uses pnpm + Turbo. Apps live in `apps/*`, shared packages in `packages/*`.
